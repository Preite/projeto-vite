Introdução

A linguagem C foi criada por Dennis Ritchie em 1972.

Ela é amplamente utilizada no desenvolvimento de sistemas.

#include <stdio.h>

Possui alta performance e controle de memória.

  • Rápida
  • Portável
Variáveis

Variáveis armazenam valores na memória.

Devem ser declaradas antes do uso.

int idade = 20;

Existem diversos tipos de dados.

  • int
  • float
Estruturas de Controle

Permitem controlar o fluxo do programa.

As mais comuns são if, else e switch.

if(x > 0){}

Também existem estruturas de repetição.

  • for
  • while
Funções

Funções organizam o código.

Facilitam reutilização e manutenção.

void mensagem(){}

Podem receber parâmetros e retornar valores.

Vetores e Matrizes

Vetores armazenam vários valores.

Matrizes representam tabelas de dados.

int vetor[10];

São estruturas fundamentais em programação.

Ponteiros

Ponteiros armazenam endereços de memória.

São recursos avançados da linguagem C.

int *p;

Permitem manipulação eficiente da memória.
